# silverwork



## 구조

### data_analysis
- 데이터 분석 모듈
- 상수 및 분석, 테스트 스크립트

### data_collection
- 데이터 수집 모듈
- 상수 및 수집, 테스트 스크립트

### resouce
- 모듈이 공용으로 사용하는 자원정보
- CSV, XML, HTML 등이 저장된다


## 실행

1. 프로젝트 checkout

2. 루트 경로(silverwork/) 에서 shell 실행후 가상환경 생성
```
python -m venv venv
source venv/bin/activate
```
3. 패키지 다운로드
```
pip install -r requirements.txt

```
